# Student Management System

## Overview
The Student Management System is a frontend web application designed to manage and organize student data efficiently. It provides a clean and responsive interface for viewing, adding, editing, and managing student records in a structured way.

## Features
- View list of students in a structured table format  
- Add new student records  
- Edit existing student information  
- Delete student records  
- Responsive and user-friendly UI  
- Clean and reusable components  

## Tech Stack
- JavaScript (ES6+)  
- HTML5  
- CSS3 
- DB JSON mock server
